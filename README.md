### 2. مشروع Algorithm-Visualizer (الميزة المراد دمجها 1)

| الخاصية                       | القيمة                                                  | ملاحظات                                                        |
|------------------------------|---------------------------------------------------------|----------------------------------------------------------------|
| التقنيات الأساسية            | React 16.14, Redux, Create React App (CRA)              | مشروع قديم يعتمد على إصدارات قديمة من React و Redux.            |
| الهدف                        | عارض خوارزميات تفاعلي                                   | ميزة قيمة جداً لمنصة تعليمية.                                  |
| استراتيجية الدمج             | دمج عرض الخوارزميات                                      | نظراً للاختلاف الكبير في إصدارات React والتبعيات، فإن أفضل طريقة هي محاولة استخراج منطق العرض (Renderers) ومنطق الخوارزميات (Tracers) وإعادة كتابتها أو تغليفها (Wrap) كمكونات React حديثة متوافقة مع React 19، أو دمجها كـ iframe إذا كان التغليف صعباً جداً. سنبدأ بمحاولة التغليف/إعادة الكتابة.  |

### 3. مشروع ProjectHtml--main (الميزة المراد دمجها 2)

| الخاصية                       | القيمة                                                  | ملاحظات                                                     |
|------------------------------|---------------------------------------------------------|-------------------------------------------------------------|
| التقنيات الأساسية            | HTML, CSS, JavaScript (Vanilla)                         | مجموعة من صفحات HTML الثابتة (Static HTML Pages) التي تبدو كدروس تعليمية أو مراجع. |
| الهدف                        | محتوى تعليمي ممتاز يمكن إضافته كجزء من مسارات التعلم في CodeLearn. | دروس ومراجع حول HTML و CSS.                                 |
| استراتيجية الدمج             | دمج المحتوى (Content Integration)                        | بما أن CodeLearn هي منصة React، لا يمكننا دمج صفحات HTML الثابتة مباشرة. سنقوم بالتالي: <br> 1. إنشاء صفحة أو مكون React جديد في CodeLearn (مثلاً Reference.tsx) لعرض المحتوى الثابت، مع تعديل الروابط الداخلية لتعمل بشكل صحيح. <br> 2. نقل محتوى HTML و CSS و JS من ProjectHtml--main إلى مجلد public في CodeLearn. <br> 3. استخدام مكون iframe داخل صفحة Reference.tsx لضمان عدم تداخل الأنماط (CSS) والسكريبتات (JS) القديمة مع بيئة React الحديثة. |
